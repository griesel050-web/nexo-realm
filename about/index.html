About

An ecosystem,
not a single app.

Nexo Realm exists because good ideas rarely arrive one at a time. It's the constant underneath a growing set of tools, apps, and experiments — the one address that never changes even as everything built on top of it does.

The vision

Every new Nexo project gets a home here from day one — whether it's a full web app, a small utility, or something still taking shape. Nexo Realm is the map; the projects are the territory.

How it's built

Static HTML, CSS, and JavaScript, driven entirely by JSON data files. No build step, no framework lock-in — just a site that's as easy to extend as it is to load.

Principles

What stays true as the realm grows

Real data only

If a project, stat, or account isn't real yet, it doesn't appear here. Honest “coming soon” beats a filled-in placeholder every time.

Everything editable in JSON

Projects and socials live in data files, not markup. Adding to the ecosystem should never mean rewriting a page.

Built to last

Free-tier infrastructure, lightweight code, and no dependencies that could disappear tomorrow.
