Every card below is generated from /data/socials.json. Nothing is invented — accounts show up here only once they've actually been added to that file.

No social accounts added yet.

This page reads directly from /data/socials.json, and that file ships empty on purpose. To add an account, open the file and push an object into the socials array — the supported platforms are already documented at the top of the file:

{
  "platform": "github",
  "handle": "@your-handle",
  "url": "https://github.com/your-handle",
  "icon": "github"
}

Supported platforms: Discord, GitHub, YouTube, TikTok, Instagram, Facebook, X, Reddit, LinkedIn, Twitch, Email, and Website.
